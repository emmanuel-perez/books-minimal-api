# Books Minimal API
(Created by emmanuel-perez)

## Stack
- .NET Core 8.0
- Minimal APIs
- C#

## Clone the repository
HTTPS
```bash
git clone https://github.com/emmanuel-perez/books-minimal-api.git
```

## Run the application
Install nuget packages
```bash
dotnet restore
```
Run the application
dotnet run

## Endpoints

### Postman Documentation:
[Docs](https://documenter.getpostman.com/view/17549938/2sB2x5Gsc4)

### Get All Books
method: GET
endpoint: {{base_url}}/api/books

### Get Book By Id
method: GET
endpoint: {{base_url}}/api/books/{id}

### Create Book
method: POST
endpoint: {{base_url}}/api/books

### Update Book
method: PUT
endpoint: {{base_url}}/api/books

### Deactivate Book
method: DELETE
endpoint: {{base_url}}/api/books
